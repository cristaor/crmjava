Automatic install from Windows Update
Open http://windowsupdate.microsoft.com, click at Check and Install Updates and choose the Security Upgrade of 4/March/2002: This update resolves the Java Applet Can Redirect Browser Traffic security vulnerability in the Microsoft java virtual machine (Microsoft VM) on Windows, and is discussed in Microsoft Security Bulletin MS02-013. Download now to prevent a maliciously crafted Java program ("applet") from silently re-routing all browser traffic to the applet's host without the user's knowledge.
